# Create Conventional Commit

Create a commit following the vault's conventional commits convention from currently staged changes.

## Commit format

```
type(scope): Message
```

- **Types**: `feat`, `fix`, `refactor`, `test`, `build`, `ci`, `docs`, `revert`
- **Message**: imperative mood, under 72 characters, in the project's working language
- **Scope**: bounded context name in lowercase, or `spec` for docs/specs, or omitted if cross-cutting
